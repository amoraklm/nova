# Contributing to Nova

Thank you for considering contributing to Nova! We welcome contributions of all kinds.

## Contribution Guidelines

1. **Fork the Repository**: Begin by forking the repository.
2. **Create a Branch**: Create a new feature branch using the naming convention `feature/your-feature-name`.
3. **Commit Changes**: Commit your changes with descriptive commit messages.
4. **Request a Review**: Open a pull request to the `main` branch and request a review from the team.

## Code of Conduct

We expect all contributors to adhere to our code of conduct. Be respectful and inclusive to everyone, regardless of background. Harassment and abusive behavior will not be tolerated.

### Reporting Issues
If you encounter any issues or have suggestions for improvements, please report them using the issue tracker on GitHub.
